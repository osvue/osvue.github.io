## VS Code

> 名称 : Visual Studio Code
>
> 开发商 : 美国微软公司
>
> 诞生时间 : 2015年4月30日Build 开发者大会
>
> 功能 : 一个运行于 `Mac OS X`、`Windows`和 `Linux `之上的，针对于编写现代 Web 和云应用的跨平台**源代码编辑器**。
