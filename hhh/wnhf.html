# 单句糊弄型

  • XX？
  • 怎么/咋就XX?
  • 太好了，谢谢**
  • 确实，该** 就XX
  • 仔细想了想，还是要XX

# 捧哏、鼓励对方说下去

  • 巧了，XX
  • 重复关键词
  • 那怎么办 / 后来呢、 原来这样啊 、为什么、 怎么会、真的吗、我都不知道诶

# 复合糊弄句

感叹词+评论事情+感受 句式：感叹词+评价事情+感受

  • 举例：
    • 对方吐槽同事, 你可以说：我晕,这人也真是,我看不惯
    • 对方埋汰者公, 你可以说：哎,这叫啥事儿啊,我也受不了
  • 感叹词："啊这/人嘛/生活嘛/大受震撼 / 难以想象/绝了/太上头了有道理/确实/我觉得也是
  • 评价： 这也太那个了吧/太过分了/咋这样啊/这叫啥事儿啊/好问题/好家伙/我觉得也是/这人也真是因崔斯特/ nice / omg / 好事多磨
  • 感受： 我无语/我气到/我烦死/我受不了/我看不惯/我不好说
  • 情绪：
    • 替你× x ：不能忍啊,过分了,替你生气了
    • 开心：
      • 跟着屏幕都能感受到你的快乐 、快乐因子穿过屏幕,感觉我也飘了、 沾沾喜气/吸吸快乐/蹭蹭好运/快乐喷雾,滋滋滋你/他也太会 xxx 了吧,我直接开心一整天
    • 生气：
      • 火猫三丈 、我鱼尾纹都气出来了别拦我,准备生气了啊

# 心里不爽

  1. 被否定/怀疑

    • 我喜欢 xx / xx 挺适合我的

  2. 被羞辱

    • 如果我是你× x 这么优秀,我才不会 xXx
    • 仗着自己年轻就可以为所欲为了吗

  3. 借东西

    • 不方便给,你懂的

  4. 不想回答/不知道怎么回答/别人提问

    • 扔回去:这是个好问题,你是怎么看的
    • 辨证学:应该辩证地来看,有好有坏,看你更注重哪方面
    • 懂学: 不方便说,你懂的/懂的都懂
    • 也学: 也对/也是/也不多/他一样
    • 其他 仔细想了一下,还是要 xx 、还行+有的好+有的不好 、 确实,该× x 就 Xx

  5. 被去哪/干嘛

  • 我去拯救地球
  • 去建设社会主义先行示范区
