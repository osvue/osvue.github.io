# 小题大做-共情-哄

# 第一步，小题大做

解决问题。能帮她解决问题就立刻表达 愿意不顾一切照顾她的姿态，小题大做，占领道德高地，从气势上稳住她。 如果生病，姨妈痛: 疼的厉害吗？我现在打车去陪你。我买了明天过去的高铁票。 如果破皮，小事我帮你在饿了么上买了药了，等等外卖就到。 如果只是心情不好，说不需要你过去，那你等我一下，我给你点完好吃的外卖，就来好好听听我的宝贝儿是哪里不开心了。

# 总之一定要在第一时间把姿态摆出来，就能稳住她。

# 第二步，倾听共情。

大多数女生需要的其实只是第二步。第一步使用过一两次，女生也不好意思太麻烦自己的男朋友。毕竟出来作总是要还的。

共情而非规劝，附和她的一些话，稍微放大一度，不要漠不关心的往下压， 也不要夸张的放大太多让她觉得你根本就不了解她，她有那么脆弱那么作吗。 那你愿意告诉我怎么了吗？

# 错误❌示范，

别想了太多，会好的，别伤心了，没事的，我以前也这样过，最后xxx就好了，一点问题没有。 这些没有错，只是不能说的过早，要她倾诉完了再说。 先共情 你现在好辛苦，这种情况真的不容易。宝贝儿受苦了，你这样我好心疼你的。然后回顾，分析。 看他问不问，再决定要不要给建议，还是优先直接帮她解决，或者仅分析。 注意一定要站在他们的角度，一个鼻孔出气，不要告诉她，她什么地方不对。除非她问。

# 第三步哄

锦上添花。哄她的鬼话骚话皮话。这时说才到点子上。 以后我一定xxx，下次带你吃好吃的好好补一补。 说点好笑的，知道吗，心情不好的时候应该倒立，这样眼屎就不会流到嘴里(逗她)。 不要气了，好不嘛？气坏还是我心疼(落脚点永远是关心她)。

# 三步必须层层慢慢推进，不可越级。不可女生的第一层基本需求还没满足，迫切需要你，你已经在第三层说冷笑话了。

如果女生心情不好是因为你惹的。该模型同样试用，能跪针板就不要跪搓衣板，能买Gucci就不要买Armani。

  1. 先占领感情高地，小题大做，稳住她。

  2. 然后共情，回顾自己确实不应该怎么怎么样。站她那边，听她倾述，及时反思自己的问题。

  3. 最后，哄。慢慢等他气消了，再买奶茶哄她，说最近手头紧，能不能等以后再买Gucci，各种哄各种逗她开心。
