# Nginx隐藏版本号操作

  1. 定位当前nginx所使用的配置文件

  2. 编缉找到的配置文件，找到**http{}**字段址其中加入：server_tokens off;

  3. 保存退出后，重新加载nginx配置文件：

nginx -s reload
